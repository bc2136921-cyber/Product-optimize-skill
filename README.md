# Product Optimize Skill

This repository contains the `product-optimizer` skill, designed to help with comprehensive product analysis and improvement.

## Overview

The Product Optimizer skill orchestrates a multi-role session to evaluate and enhance your product from various perspectives. It simulates a team of experts working together to provide actionable insights.

## Capabilities

The skill involves the following roles/phases:

1.  **Brainstorming**: Generating creative ideas and solutions.
2.  **User Simulation**: Acting as potential users to identify pain points.
3.  **Product Management**: Defining requirements, roadmap, and strategy.
4.  **UI Design**: Suggesting interface improvements and visual enhancements.
5.  **Growth**: Developing strategies for user acquisition and retention.
6.  **Commercialization**: Analyzing market fit, pricing, and revenue models.

## Usage

This skill is intended to be used by an AI agent or within a skill-based architecture.

**Trigger Phrases:**
- "Optimize product"
- "Improve app"
- "Run a full product review"

## Installation

Clone this repository and integrate the `skill.xml` (or equivalent definition) into your agent's configuration.
